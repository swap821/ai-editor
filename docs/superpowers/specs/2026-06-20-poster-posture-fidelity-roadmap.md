# VARIANT H Fidelity Roadmap — GAGOS Live Point-Field Being

## 1. Executive Summary

The being is **structurally ~75% there**: the fused brain+spine silhouette, the canon palette, the per-phase tint shifts, the vertebra-seated tab machinery, and the reabsorption motes all exist and read correctly at rest — the gaps are almost entirely in *transient state expression and the materialized-tab/nerve treatment*, not in the core organism. The **single highest-leverage theme is the materialized-tab + umbilical-nerve language shared across postures 4/5/6/7**: in points mode the tab is forced to flat dark glass with a hair-thin filament and the point-built skin is gated OFF, so four consecutive postures all read "wrong material, weak nerve" from one root cause. The second leverage cluster is the **arrival/ignition cinematic decoupling** (P0 in posture 1) and the **whole-being destructive dissolve on every tab reabsorption** (P0 in posture 7) — both are wiring/routing bugs where the mechanism already exists but fires on the wrong signal.

## 4. Recommended Execution Order

Ordered for maximum visible fidelity per unit effort, with the two cheap wiring P0s first and the shared tab/nerve block built once before the postures that depend on it.

1. **Posture 7 dissolve fix (P0-1, S).** Highest payoff-to-effort in the whole roadmap — one config row stops the entire being from vanishing on every completed tab. Land + verify on :5173 first; it's currently the most jarring break.
2. **Posture 1 arrival bridge + ignition (P0-2 + P1-1a, M+S).** Route the existing `arrivalScalarRef` into the cloud and add `uIgnite`. This makes the being condense-and-spark on load — the first thing the operator sees every session — and `uIgnite` simultaneously delivers the posture-1 "return" relight (P2-1c) for free.
3. **Build the shared "point-built tab + umbilical" block (P0-3 + P0-4 + P1-4a + P1-4b, M cluster).** This is the central leverage move: enable `OrganPointFieldSkin` in POINTS, point-build all four edges + corner brackets, sparse interior grain, raise slab scale, lateral peer placement, thicken+brighten the umbilical. Fixes posture 4 outright and pays into 5/6/7.
4. **Posture 2 intake rings + wide roots (P0-5 + P1-2a, M+S).** Add the named signature ground-ripple intake and widen the root splay — both visible at rest from every orbit angle, so the operator sees them immediately without a turn.
5. **Posture 3 cortex-hotter (P1-3a, M).** Fix the broken `uAwaken` wiring and add the capped luminance lift so the head visibly heats on a chat turn — completes the "conversation lights the core" read.
6. **Posture 2 live status pill (P1-2b, S).** Cheap diegetic-readout completion now that phase mapping is exercised by step 5; ship the state word, hold latency for a real metric.
7. **Posture 5 seat-relative focus + spine state-pulse (P1-5a + P1-5b, M+L).** Make focus lift-in-place instead of teleport, and put metabolic state on the cord itself. Needs 2+ tabs to evaluate, so it follows the single-tab postures.
8. **Posture 7 inhale + slab-crumble (P1-7a + P1-7b, M+M).** Complete the reabsorption sequence now that the destructive dissolve is gone — brainstem breath on arrival + motes peeling off the dissolving slab.
9. **Posture 6 in-slab metrics chart + working nerve-flow (P1-6a + P1-6b, L+M).** The largest single build (point-built chart from real `metricHistory`) plus traveling working-pulse; last because the streaming line-reveal already carries the core "shows work" read, so this is enhancement on a working base.
